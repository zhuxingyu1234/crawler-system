# 概述
本项目由朱星宇和高靖东完成，主要做简易分布式爬虫系统。本项目实现了通过redis数据库的调用，调取多个爬虫，进行对网页的爬取，将爬取结果存储在mongoDB数据库中。
本项目基本完成项目目标中的基础功能，对进阶功能的使用正在探索中。

# 基础介绍
## requirements
系统依据scrapy和scrapy-redis两个开源项目完成，因此，实现该系统需要安装redis,mongoDB,以及elasticsearch服务器。

## 完成的功能
完成了项目的基本要求，但是对于代理IP池的要求，即反反爬虫的要求，由于网上提供的免费代理IP基本无法使用，因此我们并没有做。其余的功能都已完成。
